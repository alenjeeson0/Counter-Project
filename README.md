## ✨ Features

- Increase the counter with the **Increase** button
- Decrease the counter with the **Decrease** button
- Reset back to zero with the **Reset** button
- Clean, minimal UI with a coral and blanchedalmond color theme

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript

## 🚀 Getting Started

No installation needed. Just clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/your-username/counter-program.git
cd counter-program
```

Then open `index.html` directly in your browser.
